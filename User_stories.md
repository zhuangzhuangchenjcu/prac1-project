# Initial Backlog Ideas

## User Stories

1. As a student, I want to create an account so that I can manage my tasks personally.  
Priority: High
Effort:2 days

2. As a student, I want to add new tasks so that I can track my assignments.  
Priority: High
Effort:2 days

3. As a student, I want to edit or delete tasks so that I can keep my task list up to date.  
Priority: Medium
Effort:2 days

4. As a student, I want to mark tasks as completed so that I can see my progress.  
Priority: Medium
Effort:1 days

5. As a student, I want to view my tasks by due date so that I can manage my time better.  
Priority: Medium
Effort:1 days

6. As a student, I want to receive reminders for deadlines so that I do not forget my assignments.  
Priority: Low
Effort:2 days

7. As a student, I want to categorize tasks by subject so that I can organize my work.  
Priority: Low
Effort:1 days
